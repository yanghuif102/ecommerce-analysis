# 🚀 作品集部署 — 生成分享链接

两个免费方案，选一个即可。**国内推荐方案一（Gitee），速度快。**

---

## 方案一：Gitee Pages（推荐，国内秒开）

### 第1步：注册 Gitee
打开 https://gitee.com → 注册账号（用手机号就行）

### 第2步：新建仓库
1. 点右上角 `+` → `新建仓库`
2. 仓库名称填：`ecommerce-analysis`
3. 公开性选：**公开**
4. 点「创建」

### 第3步：上传文件
1. 进入仓库 → 点 `上传文件`
2. 把 `portfolio` 文件夹里的 **所有文件** 拖进去上传：
   - `index.html`
   - `img/` 文件夹（含 8 张图）
3. 提交

### 第4步：开启 Pages
1. 仓库页面 → 点 `服务` → `Gitee Pages`
2. 点「启动」
3. 等待几秒 → 获得链接：
   ```
   https://你的用户名.gitee.io/ecommerce-analysis
   ```
4. 把这个链接复制到简历里、发给HR

### 第5步（重要！）：更新内容后重新部署
每次修改 `index.html` 后，进入 Gitee Pages 页面点「更新」按钮。

---

## 方案二：GitHub Pages（全球通用）

### 第1步：注册 GitHub
打开 https://github.com → 注册账号

### 第2步：新建仓库
1. 点 `+` → `New repository`
2. Repository name 填：`ecommerce-analysis`
3. 选 **Public**
4. 点 `Create repository`

### 第3步：上传文件
1. 点 `uploading an existing file`
2. 把整个 `portfolio` 文件夹内容拖进去
3. 点 `Commit changes`

### 第4步：开启 Pages
1. 仓库 → `Settings` → `Pages`
2. Source 选 `main` 分支
3. 点 `Save`
4. 等待 1-2 分钟 → 获得链接：
   ```
   https://你的用户名.github.io/ecommerce-analysis
   ```
