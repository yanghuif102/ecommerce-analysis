# 电商用户行为分析系统

> 大三下学期独立完成的数据分析项目，基于 MySQL + Python + Power BI 的全流程实战。

## 📊 项目做了什么

从零搭建了一个电商用户行为分析系统：

1. **MySQL 数据库设计** — 7 张业务表，5,000 用户 / 200 商品 / 50,000 行为记录 / 10,000 订单
2. **SQL 分析** — 漏斗分析、RFM 用户分群、GMV 趋势、留存分析等 11 个核心查询
3. **Python 可视化** — pandas 数据处理 + matplotlib 生成 8 张分析图表
4. **Power BI 管理驾驶舱** — 交互式数据看板，可按日期/类目/城市下钻

## 📈 关键发现

- 支付转化率仅 65.4%，下单到支付环节流失最严重
- 19% 用户处于流失状态（947 人），建议建立流失预警机制
- 晚高峰 18-21 点占全天行为量 45%
- 母婴用品 GMV 最高，品类结构有优化空间

## 🛠 技术栈

| 层级 | 技术 |
|------|------|
| 数据库 | MySQL 8.4 |
| 分析 | SQL（窗口函数 / CTE / JOIN） |
| 处理 | Python（pandas / numpy / matplotlib） |
